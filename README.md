# jci-ui

## 安装
```
npm install jci-ui  -S
```

### 引入
```
// 导入组件库
import jciui from 'jci-ui'
import 'jci-ui/lib/jci-ui.css'

// 注册组件库
Vue.use(jciui)

格式:jci-input

实例:<jci-input v-model="value"></jci-input>

```
